# pxtv — Импорт профиля в Prizrak-Box TV

Веб-страница для передачи ссылки на подписку или YAML-конфигурации в приложение **Prizrak-Box** на Android TV. Работает через локальную сеть — телефон и TV должны быть подключены к одному Wi-Fi.

---

## Как использовать

1. В приложении Prizrak-Box на TV откройте **Import from another device** — на экране появится QR-код с адресом вида `http://192.168.x.x:port/Prizrak-BoxTVimport`
2. Откройте страницу pxtv на телефоне
3. Вставьте ссылку на подписку в поле
4. Нажмите **Сканировать TV** и наведите камеру на QR — адрес TV определится автоматически
5. Нажмите **Отправить на TV**

Если QR не сканируется — введите адрес вручную или загрузите скриншот экрана TV.

### URL-параметр `?url=`

Можно передать ссылку на подписку прямо в URL страницы — она автоматически подставится в поле:

```
https://legiz-ru.github.io/pxtv?url=https://example.com/my-subscription
```

Удобно для использования из менеджеров паролей, шорткатов или других приложений.

## Варианты использования

### Публичная версия

Готовая страница доступна по адресу:

```
https://legiz-ru.github.io/pxtv
```

Не требует установки. Работает с любого устройства в браузере.

> ⚠️ Запросы отправляются **напрямую с вашего устройства на TV** по локальной сети. Через публичный сервер никакие данные не проходят.

---

### Самостоятельный хостинг

Если вы хотите разместить страницу на собственном сервере — скачайте файл:

**[docs/index.html](https://github.com/legiz-ru/pxtv/blob/main/docs/index.html)**

#### Вариант — Nginx (один файл)

Скопируйте файл на сервер:

```bash
cp index.html /var/www/pxtv.html
```

Добавьте в конфиг Nginx:

```nginx
location /pxtv {
    alias /var/www/pxtv.html;
    default_type text/html;
}
```

Перезагрузите Nginx:

```bash
nginx -s reload
```

Страница будет доступна по адресу `https://ваш-домен/pxtv`.
